# 环境安装

```bash
git clone https://github.com/RoyceMao/2D-Ultrasound-detection-and-segmentation.git
conda create --name py36 python=3.6
conda activate py36
conda install Keras theano pygpu opencv Jupyter PIL

conda install Keras
conda install theano
conda install pygpu
conda install opencv
conda install Jupyter
conda install PIL
```
